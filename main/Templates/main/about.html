This website project is for our final year project. We decided to learn and use Python and Django Framework from scratch to build this website because we thought by using such technologies would contribute to our future professional life. Based on this thought, we decided to call our project as "Project Future".

The starting point of the project was to allow users to translate the object in the photo they upload into a language of their choice. Since we know the importance of learning a language and the difficulty of learning new words, we thought such a system would be useful. We used Cloud Vision API and Cloud Translate API to perform these functions. Although the project currently performs its intended functions, we intend to add and develop new functions in the future.

We had a lot of fun while we developing this project and it helped us to learn new vocabulary. We hope that you have fun like us and it helps you to learn new words.
